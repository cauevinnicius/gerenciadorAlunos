Menu
1) Cadastro => quando NÃO desejo prosseguir com o cadastramento, o looping se repete. 

2) Listar alunos => parece ok, é só uma listagem geral, nada mais.

3) Pesquisar aluno => se eu coloco algo que não existe, há repetição da frase que não foram encontrados alunos.

4) Editar aluno => se eu coloco algo que não existe, há repetição da frase que não foram encontrados alunos. Depois o looping não é repetido. 

5) Deletar => quando eu insiro o ID/Nome aparece a mesma frase logo abaixo. Depois, não há nenhuma amostragem dos dados do aluno e uma validação de prosseguir. Não há mensagem de "Pressione enter para voltar ao menu principal".

6) Lançar mensalidade: se eu coloco direto o ID, a frase é repetida. quando dá certo tenho que clicar dois enteres para voltar ao menu. Quando eu coloco uma palavra invés de números, dá que é inválido e volta direto pro menu principal.

7) registrar pagamento: quando dá certo, não há a mensagem de pressionar enter para voltar. 

8) listagem de mensalidades: aqui é só uma listagem geral, nada mais. Talvez fosse importante melhorar, incluindo o nome do aluno e a data de pagamento. 

9) verificar pendências: se eu coloco um ID direto q não existe, há a repetição da frase que não houve alunos encontrados com esse nome/id. Quando coloco um que existe e aparecem os nomes, o comando diz "digite o ID do aluno que deseja lançar mensalisade" - o certo seria verificar se tem pendências. 

10) editar mensalidade: ao editar a mensalidade selecionada, há um bug e o programa quebra. Acredito que seja relacionado ao data de pagamento. 

11) exclusão: mesma coisa que os demais quando coloco direto um ID, aparece duas vezes a mesma frase. Não deixa o usuário digitar novamente. Quando eu coloco um ID, aparece só o nome do aluno e não as suas mensalidades - não tem como saber qual é o ID da mensalidade para excluir. Também não tem uma dupla validação para confirmar se deseja realmente excluir.
